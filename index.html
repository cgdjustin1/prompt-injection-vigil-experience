Methodology / Current Scope

  • 使用 cloned repo 的 Vigil YARA signatures
  • 本地离线扫描，不上传 prompt
  • 生成 JSON + CSV + 静态 HTML 审计报告
  • 报告包含 source provenance、score factors、evidence、business impact、safer version
  • 适合课堂、workshop、上线前 prompt sanity check

Limitations

  • 不是实时 LLM firewall
  • 尚未接入 transformer classifier / vector DB
  • YARA 规则不能覆盖所有 prompt injection
  • 结果应用作风险提示和审计起点，不是绝对安全判定
